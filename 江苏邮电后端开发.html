3、 实体类为未来参与排序等集合的操作操作，一般要重新哪些方法？有哪些注意点？
实现 Comparable 接口，重写 compareTo,
注意点：

  • x.compareTo(y) == -y.compareTo(x)，若抛异常则同时抛
  • 建议满足 (x.compareTo(y)==0) == (x.equals(y))

4、 Java Serializable 的作用, Equals ，== 的区别？

  • Serializable：序列化接口，实现此接口则表示支持序列化
  • Equals ，== 的区别
    • equals 是 Object 类的一个方法，==是符号
    • equals 判断对象是否满足人为定义的相等条件，==判断是否同一对象

7、 数据库访问的时候，在一个项目中是否可以同时使用 Mybatis，JPA，JDBC 等多种持久化技术？如可以，事务如何处理？
